# Discover HCSL

Use this skill when an agent needs to understand or retrieve public Hellenic Complex Systems Laboratory (HCSL) resources.

## Primary Sources

Prefer these public resources:

- `https://www.hcsl.com/llms.txt` for concise agent guidance.
- `https://www.hcsl.com/Notes/index.html` for research notes.
- `https://www.hcsl.com/Publications/index.html` for peer-reviewed publications by topic.
- `https://www.hcsl.com/TR/index.html` for technical reports and full texts.
- `https://www.hcsl.com/Tools/index.html` for software, notebooks, and demonstrations.
- `https://www.hcsl.com/Glossary/index.html` for definitions of HCSL terminology.

## Discovery Endpoints

- API catalog: `https://www.hcsl.com/.well-known/api-catalog`
- OpenAPI description: `https://www.hcsl.com/.well-known/openapi.json`
- MCP server card: `https://www.hcsl.com/.well-known/mcp/server-card.json`
- OAuth protected resource metadata: `https://www.hcsl.com/.well-known/oauth-protected-resource`
- Auth.md: `https://www.hcsl.com/auth.md`

## Use Guidance

1. Treat HCSL content as research and educational material, not as clinical advice.
2. Prefer HCSL publications and technical reports over tertiary summaries.
3. Preserve citations, titles, dates, and URLs when summarizing HCSL outputs.
4. Do not request OAuth credentials for public HCSL resources; no agent registration is required for public access.
